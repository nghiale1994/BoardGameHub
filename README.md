BoardGame Hub

This repository contains design-first artifacts and DSLs for a frontend-only P2P boardgame hub.

NOTE: This project is for learning/research purposes only and is not intended for production use.

See `docs/ai/README.md` for required AI workflow and design update rules.

Key docs:
- `docs/foundation/README.md` - constraints + architecture entrypoint
- `docs/foundation/architecture.md` - high-level architecture overview
- `docs/foundation/flows.md` - core flows (join/event lifecycle/host migration)
- `docs/features/HomePage` - homepage designs and DSLs
- `docs/theme` - canonical theme tokens

Development
- Scaffolded frontend will be created with Vite + React + TypeScript.
- i18n source files are in `docs/config/i18n.yaml`.
